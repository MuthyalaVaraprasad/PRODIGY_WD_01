# 🍽️ PRODIGY_WD_01 – Restaurant Landing Page

A modern, responsive, and visually appealing Restaurant Landing Page built using React, TypeScript, and Vite. This project was developed as part of the Prodigy InfoTech Web Development Internship.

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/MuthyalaVaraprasad/PRODIGY_WD_01.git
```

Navigate to project folder:

```bash
cd PRODIGY_WD_01
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm start
```

Build project:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
